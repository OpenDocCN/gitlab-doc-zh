## 下载

### Docker

```
docker pull apachecn0/gitlab-doc-zh
docker run -tid -p <port>:80 apachecn0/gitlab-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g gitlab-doc-zh
gitlab-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
